# Bike Rush

A mouse-controlled top-down bike traffic game built with plain HTML, CSS, and JavaScript.

## Play

Open `index.html` in a browser.

## Controls

- Move the mouse to steer the bike.
- Click to start or restart.
- Click during a ride to use boost when it is ready.

## Goal

Dodge traffic, collect coins, and survive as long as possible while the road gets faster.
